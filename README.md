# Zen Bubble Pop 🫧

A web-based stress relief game that uses computer vision (MediaPipe) to let you pop bubbles using real-world hand gestures.

## Features
- **Hands-Free:** Uses your webcam to track finger pinch gestures.
- **Zen Experience:** Soft color palettes and satisfying "pop" sound synthesis.
- **Responsive:** Optimized for mobile and desktop browsers.

## How to Run
1. Clone this repository.
2. Serve the folder using a local server (e.g., `npx serve` or VS Code Live Server) because webcam access requires `HTTPS` or `localhost`.
3. Open `index.html` in your browser.
4.If you don't want to deploy it yourself, you can also directly visit this website https://bubblepop-t7s4d017i.maozi.io
## Technologies
- [MediaPipe Hands](https://mediapipe.dev/) for real-time hand tracking.
- Web Audio API for custom synthesized pop sounds.
- HTML5 Canvas for high-performance rendering.
